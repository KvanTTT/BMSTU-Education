Здесь находятся драйвер (Driver) и оболочка для него (Shell) для USB-фильтра для любых устройств.
Логгирование происходит в файлы в папке Logs
Все проекты компилились на Visual Studio 2010.

Для того, чтобы иметь возможность компилить драйвер в студии, требуется VisualDDK.
Скачать можно здесь:
http://visualddk.sysprogs.org/download/VisualDDK-1.5.exe

Более подробную инфу смотрите в РПЗ, ТЗ и Презентации.
